# Music Telegram Bot (PHP)

Telegram music search bot - PHP version with MySQL.

## Setup

1. Upload files to hosting
2. Configure `.env` file
3. Visit `bot.php` to start

## Requirements

- PHP 8.0+
- MySQL
- PDO extension

## Files

- `bot.php` - Main bot script
- `config.php` - Configuration
- `db.php` - Database class
- `functions.php` - Helpers & Scraper
- `.env` - Environment variables
